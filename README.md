# Themes Tales

A simple blog website where users can read posts and filter them by tags.

## Demo

https://themetales.vercel.app/?page=1

## Features

- Search posts by title or body
- Filter posts by tags
- Pagination

## Tech Stack

- React
- Vite
- Ant Design
- Axios
- React Router
